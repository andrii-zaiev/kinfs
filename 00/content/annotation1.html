     Аннотация курса

     Данный курс посвящен ознакомлению с таким классом информационных систем как корпоративные информационные системы.

Для изучения курса необходимы знания таких материалов:

     - теория множеств (дисциплина «Компьютерная дискретная математика»);

     - алгоритмы и структуры данных (дисциплина «Основы программирования»);

     - принципы объектно-ориентированного подхода («Объектно-ориентированное программирование»);

     - язык запросов SQL, теория проектирования реляционных баз данных («Базы данных»)

     В курсе рассматриваются вопросы проектирования информационных систем корпоративного типа на основе СУБД Oracle, изучение языка PL/SQL, ознакомление с современными средствами, которые применяются для разработки клиент-серверных информационных систем, приобретения практических навыков проектирования, разработки и эксплуатации частей информационных систем.

     В ходе изучения дисциплины должны быть приобретены знания и умения разработки схем данных информационных систем корпоративного типа; разработки и сопровождения информационных систем; разработки программных единиц клиент-серверных баз данных на основе СУБД Oracle.
