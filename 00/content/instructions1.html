     Порядок изучения предлагаемых материалов

     Во вступительной части дисциплины рассматриваются различные классификации информационных систем и выделен класс информационных систем корпоративного уровня, которые являются распространенными для поддержки крупных организаций, требующих больших объемов данных. Отдельное внимание уделено различным классам корпоративных систем, таким как MRP, ERP, CRM, CSR системы, особенностям их внедрения и возможностям применения, а также системам электронного документооборота, которые получают все большее распространение.

     В качестве ядра корпоративной информационной системы используется СУБД корпоративного уровня, каковой и является СУБД Oracle. Второй раздел дисциплины содержит обзор логической и физической организации СУБД Oracle. Рассмотрены физическая и логическая структуры базы данных Oracle, структуры памяти и процессы в Oracle. Этот раздел помогает понять основные концепции Oracle и основы функционирования этой СУБД. Отдельное внимание уделяется управлению памятью и средствам контроля и управления хранением данных. Вопросам реализации структур хранения с управлением памятью посвящены практическая работа №1 и лабораторная работа №1.

     Третий раздел дисциплины содержит сведения о языках программирования четвертого уровня, применяемых для реализации серверных частей информационных систем на основе технологии "клиент-сервер". Особое внимание уделено процедурному расширению языка SQL фирмы Oracle - языку PL/SQL. Рассматриваются особенности создания процедур и функций, пакетов, а также специфические структуры для обработки данных - курсоры и триггеры - программные единицы, которые позволяют реализовать значительную часть обработки данных на сервере. После этого теоретического раздела целесообразно выполнение практики (2, 3) по написанию соответствующих программных единиц и проверка разработанных фрагментов в ходе выполнения лабораторных работ №2,3.

     Последний содержательный модуль посвящен организации распределенных систем хранения информации и вопросам доступа к данным, доступности и надежности данных в распределенных информационных системах. В этом разделе предлагается ознакомиться с таким аспектами реализации информационных систем, как управление параллелизмом, восстановление данных и технологиями распределения и тиражирования данных.

     Для выполнения лабораторных работ рекомендуется наличие установленного Oracle 8 Server (или любой другой версии не ниже 8.0), например, зарегестрировавшись на официальном сайте компании Oracle, можно скачать свободно распространяемое программное обеспечение Oracle Database Express Edition 11gRelease 2. Наличие утилиты SQL*Plus (или SQL*Worksheet) позволит выполнять задания в режиме командной строки, а наличие веб-интерфейса (версии 10g, 11g) улучшит визуализацию результатов.

     Практические работы можно выполнять с применением компьютеров для проверки работоспособности написанных фрагментов программ.
